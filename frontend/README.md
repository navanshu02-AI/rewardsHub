# RewardsHub Frontend

The RewardsHub frontend is a React 19 application bootstrapped with Create React App and customized via CRACO.

## Prerequisites

- Node.js 18+
- Yarn 1.x (`npm install -g yarn`)

## Available Scripts

Run these commands from the `frontend` directory:

### `yarn start`

Starts the development server and proxies API requests to the backend when configured.
Open [http://localhost:3000](http://localhost:3000) to view the app in the browser.

### `yarn test --watchAll=false`

Runs the test suite once in CI-friendly mode.

### `yarn build`

Builds the production bundle in the `build/` directory.

### `yarn eject`

Ejects the Create React App configuration. **This is a one-way operation.**
